# app-contato-seguro

## Configuração do projeto

### Instalar vue 3
```
npm install -g @vue/cli
```

```
npm install
```

### Executar projeto para desenvolvimento
```
npm run serve
```

### Executar projeto para produção
```
npm run build
```


### Personalizar configuração
See [Configuration Reference](https://cli.vuejs.org/config/).
